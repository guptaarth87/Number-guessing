API Documentation

Authentication

POST /signup: Create a new user.

Request Body:


{
  "name": "string",
  "password": "string",
  "email": "string"
}

Sample Request:


POST /signup
Content-Type: application/json

{
  "name": "John Doe",
  "password": "password123",
  "email": "john@example.com"
}

POST /login: Authenticate user.

Request Body:


{
  "email": "string",
  "password": "string"
}

Sample Request:


POST /login
Content-Type: application/json

{
  "email": "john@example.com",
  "password": "password123"
}

User Information

GET /user/:email: Get user details by email.

Request Parameters:


{
  "email": "user@example.com"
}

Sample Request:


GET /user/john@example.com

Leaderboard

GET /top10Users: Get the top 10 users with the highest max scores.

Sample Request:


GET /top10Users

Score Update

POST /updateMaxScore: Update the user's max score.

Request Body:


{
  "email": "string",
  "newMaxScore": number
}

Sample Request:


POST /updateMaxScore
Content-Type: application/json

{
  "email": "john@example.com",
  "newMaxScore": 100
}
