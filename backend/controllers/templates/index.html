API Documentation

Authentication

POST /auth/signup: Create a new user.

Request Body:

      
        {
          "name": "string",
          "password": "string",
          "email": "string"
        }

Sample Request:

      
        POST /auth/signup
        Content-Type: application/json

        {
          "name": "John Doe",
          "password": "password123",
          "email": "john@example.com"
        }

POST /auth/login: Authenticate user.

Request Body:

      
        {
          "email": "string",
          "password": "string"
        }

Sample Request:

      
        POST /auth/login
        Content-Type: application/json

        {
          "email": "email12@gmail.com",
          "password": "password123"
        }
